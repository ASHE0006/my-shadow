## 游戏简介
**作品名称：SHADOW**  
这是一款灵感来源于众多解密制单机游戏的2D RPG冒险游戏，玩家扮演一位失忆的女主人公Ashe，在时间循环的黑暗世界中，通过战斗、解谜和探索揭示隐藏的秘密。玩家需要通过操作角色收集资源（如“残损的齿轮”），提升技能，与各种敌人战斗，并最终击败Boss“Shadow”以逃出生天。

**亮点内容：**
1. **核心玩法：** 战斗结合了弹幕机制（Bullet Hell）和资源管理，同时融入了解谜与探索要素。
2. **多样化场景：** 游戏包含四大场景（钟楼、图书馆、修道院走廊、教堂礼拜堂），每个场景设计独特，融合了战斗、跑图和谜题元素。
3. **创意机制：**
   - **AI生成谜题：** 游戏在解谜环节中引入了由AI生成的问题，玩家需要通过回答问题获得继续通关的机会。
   - **像素级碰撞检测：** 精确还原交互与战斗细节。
4. **战斗机制：** 非进攻类的交互方式，与后续飞书转变为友好NPC与玩家进行交易的情节形成了巧妙的呼应;同时进行躲避ghost攻击，定位魔法弹发射位置，发射魔法弹的三重行为，多按键的同时运用也成为了游戏战斗的难点所在。

## 游戏教程
### 游戏安装与运行
1. **环境要求：**
   - 操作系统：Windows / MacOS
   - 必备环境：Python 3.x，Pygame库，openai库，numpy库

2. **运行步骤：**
   - 下载项目文件并解压。
   - 进入项目目录，运行 `main.py`：
     ```bash
     python main.py
     ```
   - 开始游戏！

### 游戏玩法
- **基础操作：**
  - `W/A/S/D` 控制角色上下左右移动。
  - `Z` 进行对话。
  - `N` 切换场景或开门。
  - 空格键：发射弹幕攻击。
  - 鼠标指针：控制攻击方向。
  - `M` 脱离卡死
  - `F`：特殊交互（如阅读飞书/收集场景掉落物）。
- **目标：**
  - 在场景中收集“残损的齿轮”作为货币，提升角色能力。
  - 解锁场景谜题，击败敌人并最终挑战Boss“Shadow”。
  - 游戏胜利条件：击败最终Boss，完成逃脱。

## 场景介绍
1. **钟楼（初始场景）：** 玩家出生地，散布有齿轮道具，需收集完成目标以解锁通往图书馆的路径。
2. **图书馆：** 寻找三本特定书籍来解锁隐藏的暗门，同时需要躲避或击败飞书敌人。
3. **修道院走廊：** 包含追逐与解谜环节，玩家需躲避怪物或者杀死怪物，以及回答AI生成的谜题来继续探索。
4. **教堂礼拜堂（最终场景）：** 玩家需与最终Boss“Shadow”决战。

---

## 核心机制
1. **战斗：** 弹幕战斗机制，玩家与怪物都可发射弹幕攻击。
2. **解谜：** 融入AI生成谜题，答对才能推进剧情。
3. **资源管理：** 收集硬币以升级技能或购买道具，血量资源需合理分配。
4. **碰撞系统：** 基于Pygame像素级碰撞检测实现精准交互体验，并具备脱离卡死功能。
5. **菜单设计：** 包括主菜单、商店界面与胜利界面。

---

## 角色与敌人
- **玩家角色：** 女主人公Ashe，可通过升级提升技能与属性。
- **友好NPC：**
  - **对话NPC**：提供游戏提示与背景信息。
  - **交易NPC**：通过齿轮货币购买技能和道具。
- **敌人：**
  1. **飞书**：随机飞行，玩家需在其掉落后靠近阅读消灭。
  2. **骷髅（Monster）**：会追踪玩家并发射弹幕攻击。
  3. **幽灵（Ghost）**：触碰玩家后触发解谜环节，回答问题来摆脱威胁。
- **最终Boss：** “Shadow”，需综合运用所有技能与资源击败。

## 实现简介
### 项目结构
- **核心模块：**
  - `Main.py`：管理游戏主逻辑、场景切换与碰撞检测。
  - `Main2.py`：管理战斗场景主逻辑、玩家血量与攻击方式。
  - `Player.py`：定义玩家角色的属性与行为。
  - `NPC.py`：定义NPC与怪物的行为逻辑。
  - `Dialog.py`：实现对话框。
  - `Scene.py`：定义场景逻辑与渲染。
- **其他模块：**
  - `Settings.py`：存储游戏参数配置。
  - `Map.py`：定义地图与物品。
  - `block.py`：定义障碍物。
  - ...

### 技术亮点
- **像素级碰撞检测：** 提高交互精准度与公平性。
- **弹幕机制：** 增加战斗趣味性与挑战性。
- **AI谜题生成：** 动态生成谜题，提高游戏随机性与智力挑战。
